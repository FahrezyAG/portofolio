Introduction

A website for anime that retrieves data from an API. This anime website features a search function, GitHub-based login, commenting, and the ability to add anime to a personal list. Additionally, users can search for anime based on genres. The website also provides information on ongoing anime, keeping users informed about the latest releases.

This project is an assignment from the animelist playlist of dea afrizal's youtube channel.

From this assignment I got a lot of insight about frontend
